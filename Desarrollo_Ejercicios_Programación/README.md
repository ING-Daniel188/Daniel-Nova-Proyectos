# Proyecto de Programación 

## Descripción del proyecto
Este proyecto reúne los ejercicios, prácticas y trabajos realizados en la materia de Programación. Sirvió como apoyo para aplicar los conceptos vistos en clase y mejorar la forma de pensar y resolver problemas mediante código.

El enfoque principal no fue solo que los programas funcionaran, sino también entender cómo y por qué funcionan, reforzando la lógica, el orden y la claridad al programar.

## Objetivo
Poner en práctica los conocimientos adquiridos y desarrollando soluciones a distintos problemas propuestos en clase.

Además, el proyecto busca evidenciar el progreso desde conceptos básicos hasta ejercicios un poco más complejos, mostrando la evolución del razonamiento lógico.

## Temas abordados
En los archivos del proyecto se trabajan temas como:

- Uso de variables y tipos de datos
- Estructuras condicionales
- Ciclos y repeticiones
- Funciones o métodos
- Resolución de problemas mediante algoritmos
- Organización básica del código

Cada archivo representa un ejercicio o actividad relacionada con los temas vistos en la asignatura.

## Organización del proyecto
La carpeta contiene los diferentes archivos creados durante el semestre. Los nombres de los archivos permiten identificar fácilmente el propósito de cada uno, ya sea un ejercicio práctico, un taller o una actividad evaluativa.

La estructura es sencilla, pensada para facilitar la lectura y comprensión del código.

## Herramientas utilizadas
- Lenguaje de programación java
- Entorno de desarrollo utilizado eclipse
