# Proyecto de Programación – 2023-2

## Descripción general
Este proyecto corresponde a los trabajos y ejercicios desarrollados durante el semestre 2023-2 en la asignatura de Programación. El objetivo principal de este proyecto fue poner en práctica los conceptos vistos en clase, fortaleciendo la lógica de programación, la resolución de problemas y el uso correcto de estructuras básicas del lenguaje.

A lo largo del desarrollo del proyecto se buscó no solo que el código funcionara, sino también que fuera claro, ordenado y entendible, aplicando buenas prácticas aprendidas durante el curso.

## Objetivo del proyecto
El propósito de este proyecto es demostrar el aprendizaje progresivo en programación, abordando problemas de diferente nivel de complejidad y aplicando conceptos como:

- Pensamiento lógico
- Análisis y planteamiento de problemas
- Implementación de soluciones mediante código
- Organización del proyecto en archivos y carpetas

## Contenidos trabajados
Dentro de este proyecto se pueden encontrar ejercicios y desarrollos relacionados con temas como:

- Variables y tipos de datos
- Condicionales
- Ciclos
- Funciones o métodos
- Manejo básico de datos
- Resolución de problemas prácticos

Cada parte del proyecto representa una etapa del aprendizaje durante el semestre.

## Estructura del proyecto
La carpeta del proyecto contiene los archivos desarrollados durante el curso, organizados de manera que sea fácil identificar cada ejercicio o actividad. Los nombres de los archivos reflejan el tema o el objetivo de cada uno.

## Tecnologías utilizadas
- Lenguaje de programación java
- Entorno de desarrollo utilizado eclipse
