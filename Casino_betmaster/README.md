## Descripción del proyecto
Este proyecto llamado **Casino BetMaster** consiste en una simulación de un casino desarrollada como parte de un trabajo académico de programación. El objetivo principal fue aplicar los conocimientos aprendidos en clase para crear un sistema que represente el funcionamiento básico de un casino, incluyendo apuestas, decisiones del usuario y resultados aleatorios.

El proyecto se desarrolló con un enfoque práctico, buscando entender cómo estructurar un programa, manejar la lógica del juego y permitir la interacción con el usuario de forma clara y ordenada.

## Objetivo
El objetivo de este proyecto es reforzar la lógica de programación mediante la creación de un juego tipo casino, donde se ponen en práctica conceptos fundamentales como condiciones, ciclos, manejo de datos y generación de resultados aleatorios.

Además, el proyecto busca simular situaciones reales de apuestas de manera sencilla, sin llegar a ser un sistema complejo o comercial, sino con fines netamente educativos.

## Funcionamiento general
El programa permite al usuario interactuar con el sistema de casino, tomar decisiones relacionadas con apuestas y recibir resultados según la lógica implementada. Cada acción del usuario genera una respuesta del sistema, lo que ayuda a comprender cómo fluye un programa desde la entrada de datos hasta la salida de resultados.

El comportamiento del juego está basado en reglas simples, pensadas para facilitar el aprendizaje y la comprensión del código.

## Conceptos aplicados
En este proyecto se aplican principalmente los siguientes conceptos de programación:

- Uso de variables y tipos de datos
- Estructuras condicionales
- Ciclos de repetición
- Manejo de entradas y salidas
- Lógica para simulación de juegos
- Organización básica del código

## Organización del proyecto
La carpeta del proyecto contiene los archivos necesarios para ejecutar la simulación del casino. La estructura es sencilla y está pensada para que cualquier persona que revise el código pueda entender fácilmente qué hace cada parte del programa.

Los nombres de los archivos reflejan su función dentro del proyecto.

## Herramientas utilizadas
- Lenguaje de programación Java
- Entorno de desarrollo utilizado Eclipse
