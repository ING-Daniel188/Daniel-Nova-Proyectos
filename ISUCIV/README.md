## Descripción del proyecto
ISUCI es un proyecto académico desarrollado en Java, enfocado en la gestión y simulación de información relacionada con el ciclismo, tomando como referencia el contexto de la UCI (Unión Ciclista Internacional).

Esta versión representa una mejora respecto a la primera entrega del proyecto, tanto en organización del código como en la implementación de nuevas funcionalidades y un manejo más claro de la programación orientada a objetos. El objetivo principal fue aplicar de manera más sólida los conceptos vistos en clase y corregir o ampliar aspectos del sistema inicial.

## Objetivo
El objetivo del proyecto ISUCI es fortalecer el uso de la programación orientada a objetos mediante un sistema más estructurado y completo, mejorando la lógica, la organización del código y la separación de responsabilidades.

Además, esta versión busca demostrar la evolución del proyecto y del aprendizaje del estudiante a lo largo del curso.

## Funcionamiento general
El sistema permite gestionar información relacionada con ciclistas y competencias, controlando los datos a través de clases bien definidas y una lógica central que administra el flujo del programa.

ISUCI mantiene un enfoque académico, por lo que su funcionamiento se basa en reglas claras y sencillas, pensadas para facilitar la comprensión del código y no para simular un sistema real a nivel comercial.

## Mejoras respecto a la versión anterior
En esta segunda versión del proyecto se pueden notar mejoras como:

- Mejor organización de paquetes y clases
- Uso más claro de la programación orientada a objetos
- Código más ordenado y legible
- Ajustes en la lógica del sistema
- Mayor claridad en la estructura general del proyecto

Estas mejoras reflejan el proceso de aprendizaje y corrección realizado a partir de la primera versión.

## Conceptos aplicados
En este proyecto se aplican principalmente los siguientes conceptos:

- Programación orientada a objetos
- Clases y objetos
- Herencia y polimorfismo
- Encapsulamiento
- Separación de responsabilidades
- Organización de paquetes en Java
- Manejo básico de datos

## Organización del proyecto
El proyecto está organizado por paquetes, separando las clases según su función dentro del sistema. Esta estructura facilita la lectura del código y permite entender rápidamente el rol de cada componente del programa.

La organización refleja buenas prácticas básicas aprendidas durante el curso.

## Tecnologías utilizadas

- Entorno de desarrollo: Eclipse IDE
